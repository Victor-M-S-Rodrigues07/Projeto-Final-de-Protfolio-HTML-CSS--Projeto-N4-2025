## <h2> 📝 Descrição do Projeto </h2>

O presente projeto retrata uma proposta de portfólio com marcação via **HTML** e práticas de estilização com **CSS**. Foram utilizadas classes CSS para maior modularidade e praticidade na decoração, o uso de Flexbox a fim de organizar o *display* do site, entre outros métodos usados no mercado. <br>

A proposta do site foi realizada visando a aprovação e aprendizado prático do curso *HTML e CSS: Classes, posicionamento e Flexbox* (link nos créditos da documentação). <br>

## <h2> 🎯 Objetivos da Aprendizagem </h2>

- Aprenda o que são classes CSS e sua importância;
- Conheça o Flexbox e aplique técnicas para posicionar seus elementos de forma prática;
- Saiba como estilizar seus textos, fontes e ícones;
- Entenda como posicionar os elementos de uma página com CSS;
- Descubra como aplicar efeitos de estilização alterando cores, arredondamentos e importar fontes para seu projeto. <br>
